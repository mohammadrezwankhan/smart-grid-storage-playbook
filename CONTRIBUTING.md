# Contributing

Contributions should make smart-grid and storage concepts easier to inspect, teach, or reuse.

## Good Contributions

- Improve explanations.
- Add diagrams.
- Add glossary entries.
- Link to public technical references.

## Pull Request Checklist

- [ ] The explanation is technically accurate.
- [ ] Acronyms are defined.
- [ ] Diagrams include alt text or descriptions.
- [ ] Sources are public.
